# String Search & Replace Program

So basically, this program prompts the user to enter a source string (like "I HATE YOU")
and then prompts for a pattern string (like "HATE" ) , and then a replacement string to replace the pattern string.

---------------------------------------------------------------------------------------------------------------------
EXAMPLE:
source      =   "I HATE YOU"
pattern     =   "HATE"
replacement =   "LOVE

size_t position{ "I HATE YOU".find("HATE") }

'H' => index=2

if( 2 != a really big value thats impossible for any actual valid index in a string to ever be ){
      "I HATE YOU".replace(2,4,"LOVE")
}

COUT << "I LOVE YOU"
